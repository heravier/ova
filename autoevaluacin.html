Autoevaluación 1: Introducción al diseño 3D

Pregunta

¿Qué es el diseño 3D?

Respuestas

El proceso de crear representaciones bidimensionales de objetos.

El proceso de crear representaciones tridimensionales de objetos utilizando software especializado.

El proceso de crear imágenes animadas para películas.

El proceso de crear objetos físicos mediante impresoras 3D.

Pregunta

¿Qué acción se debe realizar para exportar un modelo 3D creado en Tinkercad para usarlo en un proyecto de Realidad Aumentada (AR)?

Respuestas

Hacer clic en "Imprimir" y luego seleccionar "Exportar".

Hacer clic en "Descargar" y seleccionar el formato .STL.

Hacer clic en "Compartir" y copiar el enlace.

Hacer clic en "Guardar como imagen".
 

Pregunta

 ¿Qué función tiene la herramienta "TinkerCad" dentro del contexto de este módulo?
 

Respuestas

 Permite a los estudiantes crear modelos 3D básicos de forma rápida y sencilla.

Sirve para diseñar experiencias interactivas en Realidad Virtual.

Permite crear gráficos vectoriales para impresión en 2D.
